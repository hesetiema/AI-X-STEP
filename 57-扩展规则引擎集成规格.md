# 这个扩展版覆盖了什么

## 1. 分组覆盖完整 ruleset
已经覆盖：
- `R101-R104`
- `R201-R203`
- `R301-R302`
- `R401-R402`
- `R501-R504`

## 2. 既测单规则召回，也测多规则共存
比如：
- `R102 + R103 + R501`
- `R201 + R203`
- `R202 + R203`
- `R301 + R302`
- `R401 + symptom`
- `R501 + R502 + R503 + R504`

## 3. 保持 recall-first 测试风格
没有在 integration 里验证：
- 排序第一是谁
- top cause 是谁
- symptom 是否下沉
- finalScore 大小

这些都留给：
- `RankingService`
- `DiagnosisConclusionService`
