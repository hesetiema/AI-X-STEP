下面直接给你 **DiagnosisController / e2e 测试骨架**。  
按你现在的设计前提来：

- `POST /api/v1/diagnosis`
  - 只创建任务
  - 不立即执行完整诊断
- `GET /api/v1/diagnosis/:id`
  - 先查缓存
  - 无缓存时 lazy execute
  - 执行后缓存结果
- 当前阶段 repository 用 token 注入
- 测试以 **Nest + supertest** 的 HTTP 级 e2e 为主
- 先覆盖 MVP 主链，不追求复杂 infra

# 2. 测试目标拆分

这份 e2e 骨架建议先覆盖 8 类行为：

---

## POST /api/v1/diagnosis

1. 成功创建 diagnosis task
2. 参数非法时返回 400
3. 创建时不触发 rule engine / conclude / build explanation / build domino

---

## GET /api/v1/diagnosis/:id

4. task 不存在返回 404
5. 已有缓存结果时直接返回缓存
6. 无缓存时执行 lazy diagnosis 并返回结果
7. lazy execute 后会保存结果
8. lazy execute 过程中抛错时返回 500（或你当前全局异常格式）

# 3. 先给一个最小可运行的 app factory

这个文件的目标是：

- 减少 e2e spec 里的样板代码
- 每次测试快速装配 app
- 直接 mock diagnosis module 依赖

## `src/modules/diagnosis/__tests__/e2e/test-app.factory.ts`

```ts
import { INestApplication, ValidationPipe } from '@nestjs/common';
import { Test, TestingModule } from '@nestjs/testing';

import { DiagnosisController } from '../../controller/diagnosis.controller';
import { DiagnosisCommandService } from '../../application/services/diagnosis-command.service';
import { DiagnosisQueryService } from '../../application/services/diagnosis-query.service';

export interface DiagnosisE2eMocks {
  diagnosisCommandService: {
    create: jest.Mock;
  };
  diagnosisQueryService: {
    getById: jest.Mock;
  };
}

export interface DiagnosisE2eAppContext {
  app: INestApplication;
  moduleRef: TestingModule;
  mocks: DiagnosisE2eMocks;
}

export const createDiagnosisE2eApp =
  async (): Promise<DiagnosisE2eAppContext> => {
    const mocks: DiagnosisE2eMocks = {
      diagnosisCommandService: {
        create: jest.fn(),
      },
      diagnosisQueryService: {
        getById: jest.fn(),
      },
    };

    const moduleRef = await Test.createTestingModule({
      controllers: [DiagnosisController],
      providers: [
        {
          provide: DiagnosisCommandService,
          useValue: mocks.diagnosisCommandService,
        },
        {
          provide: DiagnosisQueryService,
          useValue: mocks.diagnosisQueryService,
        },
      ],
    }).compile();

    const app = moduleRef.createNestApplication();

    app.useGlobalPipes(
      new ValidationPipe({
        whitelist: true,
        forbidNonWhitelisted: true,
        transform: true,
      }),
    );

    await app.init();

    return {
      app,
      moduleRef,
      mocks,
    };
  };
```

## `src/modules/diagnosis/__tests__/e2e/diagnosis.controller.e2e-spec.ts`

```ts
import request from 'supertest';
import {
  INestApplication,
  InternalServerErrorException,
  NotFoundException,
} from '@nestjs/common';

import { createDiagnosisE2eApp } from './test-app.factory';
import {
  buildDiagnosisTask,
  buildDiagnosisResult,
  buildDiagnosisContext,
  buildRuleFinding,
  buildDiagnosisConclusion,
  buildDiagnosisExplanation,
  buildDominoChain,
} from '../helpers';
import {
  buildClickRequestNotSentScenario,
  buildInspectApiFieldMissingScenario,
} from '../fixtures';

describe('DiagnosisController (e2e)', () => {
  let app: INestApplication;

  let mocks: Awaited<ReturnType<typeof createDiagnosisE2eApp>>['mocks'];

  beforeEach(async () => {
    const ctx = await createDiagnosisE2eApp();
    app = ctx.app;
    mocks = ctx.mocks;
  });

  afterEach(async () => {
    await app.close();
  });

  describe('POST /api/v1/diagnosis', () => {
    it('should create diagnosis task and return 201', async () => {
      const task = buildDiagnosisTask({
        id: 'diag_post_001',
        mode: 'inspect_diagnosis',
        target: {
          pageUrl: '/demo/detail',
          selector: '#amount',
        },
      });

      mocks.diagnosisCommandService.create.mockResolvedValue(task);

      const payload = {
        mode: 'inspect_diagnosis',
        target: {
          pageUrl: '/demo/detail',
          selector: '#amount',
        },
      };

      const res = await request(app.getHttpServer())
        .post('/api/v1/diagnosis')
        .send(payload)
        .expect(201);

      expect(mocks.diagnosisCommandService.create).toHaveBeenCalledTimes(1);
      expect(mocks.diagnosisCommandService.create).toHaveBeenCalledWith(
        expect.objectContaining(payload),
      );

      expect(mocks.diagnosisQueryService.getById).not.toHaveBeenCalled();

      expect(res.body).toEqual(
        expect.objectContaining({
          id: 'diag_post_001',
          mode: 'inspect_diagnosis',
        }),
      );
    });

    it('should return 400 when mode is missing', async () => {
      const payload = {
        target: {
          pageUrl: '/demo/detail',
          selector: '#amount',
        },
      };

      await request(app.getHttpServer())
        .post('/api/v1/diagnosis')
        .send(payload)
        .expect(400);

      expect(mocks.diagnosisCommandService.create).not.toHaveBeenCalled();
    });

    it('should return 400 when target contains unknown fields', async () => {
      const payload = {
        mode: 'inspect_diagnosis',
        target: {
          pageUrl: '/demo/detail',
          selector: '#amount',
          unexpectedField: 'boom',
        },
      };

      await request(app.getHttpServer())
        .post('/api/v1/diagnosis')
        .send(payload)
        .expect(400);

      expect(mocks.diagnosisCommandService.create).not.toHaveBeenCalled();
    });

    it('should only create task and should not trigger query flow on POST', async () => {
      const task = buildDiagnosisTask({
        id: 'diag_post_002',
      });

      mocks.diagnosisCommandService.create.mockResolvedValue(task);

      await request(app.getHttpServer())
        .post('/api/v1/diagnosis')
        .send({
          mode: 'click_diagnosis',
          target: {
            pageUrl: '/demo/list',
            selector: '#submit-btn',
            actionId: 'click-submit',
          },
        })
        .expect(201);

      expect(mocks.diagnosisCommandService.create).toHaveBeenCalledTimes(1);
      expect(mocks.diagnosisQueryService.getById).not.toHaveBeenCalled();
    });
  });

  describe('GET /api/v1/diagnosis/:id', () => {
    it('should return 404 when diagnosis task does not exist', async () => {
      mocks.diagnosisQueryService.getById.mockRejectedValue(
        new NotFoundException('Diagnosis task not found'),
      );

      const res = await request(app.getHttpServer())
        .get('/api/v1/diagnosis/diag_not_found')
        .expect(404);

      expect(mocks.diagnosisQueryService.getById).toHaveBeenCalledWith(
        'diag_not_found',
      );
      expect(res.body.message).toContain('Diagnosis task not found');
    });

    it('should return cached diagnosis result directly', async () => {
      const scenario = buildInspectApiFieldMissingScenario();

      const cachedResult = buildDiagnosisResult({
        diagnosisId: scenario.task.id,
        context: scenario.context,
        findings: scenario.findings,
        conclusion: scenario.conclusion,
        explanation: scenario.explanation,
        dominoChain: scenario.dominoChain,
      });

      mocks.diagnosisQueryService.getById.mockResolvedValue(cachedResult);

      const res = await request(app.getHttpServer())
        .get(`/api/v1/diagnosis/${scenario.task.id}`)
        .expect(200);

      expect(mocks.diagnosisQueryService.getById).toHaveBeenCalledWith(
        scenario.task.id,
      );
      expect(res.body).toEqual(
        expect.objectContaining({
          diagnosisId: scenario.task.id,
          status: 'completed',
          conclusion: expect.objectContaining({
            topFinding: expect.objectContaining({
              ruleCode: 'R101',
            }),
          }),
        }),
      );
    });

    it('should return lazy executed diagnosis result when cache is absent', async () => {
      const scenario = buildClickRequestNotSentScenario();

      const result = buildDiagnosisResult({
        diagnosisId: scenario.task.id,
        context: scenario.context,
        findings: scenario.findings,
        conclusion: scenario.conclusion,
        explanation: scenario.explanation,
        dominoChain: scenario.dominoChain,
      });

      mocks.diagnosisQueryService.getById.mockResolvedValue(result);

      const res = await request(app.getHttpServer())
        .get(`/api/v1/diagnosis/${scenario.task.id}`)
        .expect(200);

      expect(mocks.diagnosisQueryService.getById).toHaveBeenCalledTimes(1);
      expect(mocks.diagnosisQueryService.getById).toHaveBeenCalledWith(
        scenario.task.id,
      );

      expect(res.body).toEqual(
        expect.objectContaining({
          diagnosisId: scenario.task.id,
          status: 'completed',
          conclusion: expect.objectContaining({
            topFinding: expect.objectContaining({
              ruleCode: 'R402',
            }),
          }),
          explanation: expect.objectContaining({
            summaryText: expect.stringContaining('request 未发送'),
          }),
        }),
      );
    });

    it('should return completed result with empty findings when diagnosis finishes without matched rules', async () => {
      const result = buildDiagnosisResult({
        diagnosisId: 'diag_empty_001',
        status: 'completed',
        context: buildDiagnosisContext({
          diagnosisId: 'diag_empty_001',
          responseSuccess: false,
          storeUpdated: false,
          renderTriggered: false,
        }),
        findings: [],
        conclusion: buildDiagnosisConclusion({
          topFinding: null,
          supportingFindings: [],
          symptomFindings: [],
          diagnosisState: 'insufficient_evidence' as any,
          summary: '当前证据不足，无法确认明确根因。',
          repairHints: ['补充接口响应快照'],
          scoreBreakdown: [],
        }),
        explanation: buildDiagnosisExplanation({
          summaryText: '当前证据不足，暂时无法确认明确根因。',
          evidenceNarrative: [],
          operatorAdvice: ['建议处理：补充接口响应快照'],
          symptomNotes: [],
        }),
        dominoChain: buildDominoChain({
          mode: 'inspect_diagnosis',
          nodes: [],
        }),
      });

      mocks.diagnosisQueryService.getById.mockResolvedValue(result);

      const res = await request(app.getHttpServer())
        .get('/api/v1/diagnosis/diag_empty_001')
        .expect(200);

      expect(res.body).toEqual(
        expect.objectContaining({
          diagnosisId: 'diag_empty_001',
          findings: [],
          conclusion: expect.objectContaining({
            topFinding: null,
            diagnosisState: 'insufficient_evidence',
          }),
        }),
      );
    });

    it('should return 500 when lazy execution fails unexpectedly', async () => {
      mocks.diagnosisQueryService.getById.mockRejectedValue(
        new InternalServerErrorException('lazy execution failed'),
      );

      const res = await request(app.getHttpServer())
        .get('/api/v1/diagnosis/diag_runtime_error')
        .expect(500);

      expect(mocks.diagnosisQueryService.getById).toHaveBeenCalledWith(
        'diag_runtime_error',
      );
      expect(res.body.message).toContain('lazy execution failed');
    });
  });
});
```

## `create-diagnosis.dto.ts`

```ts
import {
  IsIn,
  IsNotEmpty,
  IsObject,
  IsOptional,
  IsString,
  ValidateNested,
} from 'class-validator';
import { Type } from 'class-transformer';

export class DiagnosisTargetDto {
  @IsOptional()
  @IsString()
  @IsNotEmpty()
  pageUrl?: string;

  @IsOptional()
  @IsString()
  @IsNotEmpty()
  selector?: string;

  @IsOptional()
  @IsString()
  @IsNotEmpty()
  actionId?: string;
}

export class CreateDiagnosisDto {
  @IsString()
  @IsIn(['inspect_diagnosis', 'click_diagnosis'])
  mode!: 'inspect_diagnosis' | 'click_diagnosis';

  @IsObject()
  @ValidateNested()
  @Type(() => DiagnosisTargetDto)
  target!: DiagnosisTargetDto;
}
```

# 6. controller 最小骨架参考

如果你还没完全落 controller，这份 e2e 对应的 controller 可以长这样：

## `diagnosis.controller.ts`

```ts
import { Body, Controller, Get, Param, Post } from '@nestjs/common';
import { CreateDiagnosisDto } from '../dto/create-diagnosis.dto';
import { DiagnosisCommandService } from '../application/services/diagnosis-command.service';
import { DiagnosisQueryService } from '../application/services/diagnosis-query.service';

@Controller('/api/v1/diagnosis')
export class DiagnosisController {
  constructor(
    private readonly diagnosisCommandService: DiagnosisCommandService,
    private readonly diagnosisQueryService: DiagnosisQueryService,
  ) {}

  @Post()
  async create(@Body() dto: CreateDiagnosisDto) {
    return this.diagnosisCommandService.create(dto);
  }

  @Get(':id')
  async getById(@Param('id') id: string) {
    return this.diagnosisQueryService.getById(id);
  }
}
```

# 9. 这套骨架覆盖的测试边界

当前这份 e2e 是 **controller HTTP 行为骨架**，不是全真模块集成。  
它验证的是：

- 路由是否通
- DTO 校验是否生效
- controller 是否调用正确 service
- HTTP 状态码是否正确
- 返回结构是否正确穿透

它**没有**验证：

- repository 真存取
- lazy execution 真正内部编排
- rule engine 真命中

这些你前面 integration spec 已经覆盖了，所以这里保持轻量是对的。
